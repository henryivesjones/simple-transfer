# easy-transfer
Easy data transfers between relational databases.

# Quickstart
See `examples/postgresql_to_postgresql.py`

# Supported Options
## Supported transfers
 - Overwrite
 - Append
 - Swap

## Supported Databases
 - PostgreSQL

## Supported temporary storage locations
 - Local File System
 - AWS S3
 - Azure Blob Storage
 - GCP Cloud Storage
